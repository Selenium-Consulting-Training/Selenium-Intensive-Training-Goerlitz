Selenium Training CheatSheet
Driver Initialization
  • FirefoxWebDriver driver = new FirefoxDriver();
  • ChromeWebDriver driver = new ChromeDriver();
  • EdgeWebDriver driver = new EdgeDriver();
  • SafariWebDriver driver = new SafariDriver();
Basic Actions
  • ClickwebElement.click();
  • SendKeyswebElement.sendKeys("Test");
  • Replace Existing TextwebElement.clear();webElement.sendKeys("Test");orwebElement.sendKeys(Keys.chord(Keys.CONTROL, "a") , "Test");
  • GetAttributeswebElement.getAttribute("class");
  • GetTextwebElement.getText();
Extended Actions
  • Crete Action Class new Actions(webDriver)
  • Perform Action new Actions(webDriver).action.action2.perform();
Alerts
  • Get Text driver.switchTo().alert().getText()
  • Accept driver.switchTo().alert()...
  • Decline driver.switchTo().alert()...
  • SendKeys driver.switchTo().alert()...
Locators
  • By IDdriver.findElement(By.id("q"))
  • By Namedriver.findElement(By.name("q"))
  • By TagNamedriver.findElement(By.tagName("span"))
  • By LinkTextdriver.findElement(By.LinkText("Weiter"))
  • By CSS-Selectordriver.findElement(By.cssSelector("#main>.menu>a"))
  • By Xpathdriver.findElement(By.xpath("//input[@id='q']"))
XPaths
  • DIV //div
  • div span //div//span
  • div > span //div/span
  • Attribute ID //*[@id="id"]
  • Input with Type submit //input[@type="submit"]
Navigation
  • To URL driver.get("http://www.selenium-consulting.ch"); or driver.navigate().to("http://www.selenium-consulting.ch");
  • Refresh Page driver.navigate().refresh();
  • Navigate forwards in History driver.navigate().forward();
  • Navigate backwards in History driver.navigate().backward();
Window
  • Get Actual Window Handle driver.getWindowHandle()
  • Get All Window Handles driver.getWindowHandles()
  • Switch Window driver.switchTo()...
